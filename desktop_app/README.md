# Solena Desktop MVP

This is the first desktop entry point for Solena.

## What it does

- opens a PyQt6 window
- lets the user choose the private core folder
- lets the user import a folder of dialogue files
- loads the Solena GPS and pipeline guide
- shows a structured JSON preview of the analysis

## What it does not do yet

- it does not call a real AI backend
- it does not generate lab modules automatically
- it does not run the full refinement pipeline

## Run

```bash
python main.py
```

## Dependencies

- Python 3.12+
- PyQt6
